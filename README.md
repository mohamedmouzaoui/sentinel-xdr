# sentinel-xdr
Sentinel XDR v3.0 is an enterprise-grade, lightweight Extended Detection and Response platform optimized for resource-constrained VMs. It unifies multi-tenant SOC management, ISO 27001 auditing, MITRE ATT&amp;CK dashboards, and Sigma rules with ML threat detection (RandomForest). Includes SOAR playbooks, CTI (STIX/MISP), and TheHive.
